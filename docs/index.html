Get Your Pet Today

It only takes a minute to set up!

1

Clone the Repository

git clone https://github.com/pegasusheavy/vscode-tamagotchi.git
2

Install & Build

cd vscode-tamagotchi
pnpm install
pnpm run compile
3

Launch in VSCode

Open the folder in VSCode and press F5 to start the Extension Development Host

# Or package it: pnpm run package
4

Meet Your Pet!

Open the Command Palette (Ctrl+Shift+P) and run "Tamagotchi: Show Tamagotchi"
